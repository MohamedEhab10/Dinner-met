What's For Dinner

A simple and interactive web application that helps users discover meal ideas for dinner.  
The project provides an intuitive interface where users can explore recipes and get daily meal inspiration.

---

## Overview

"What's For Dinner" is a front-end web project designed to demonstrate modern web development practices using HTML, CSS, Bootstrap, and JavaScript.

The application features a clean navigation bar, recipe display section, and interactive components to improve the user experience.

---

## Features

- Responsive navigation bar
- Modern UI using Bootstrap
- Recipe inspiration interface
- Interactive elements with JavaScript
- Clean and structured layout
- Mobile-friendly design

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Font Awesome

---

## How to Run

1. Download or clone the repository

2. Open the project folder

3. Run the **index.html** file in your browser.

---

## 🎯 Purpose

This project was created to practice front-end development skills including layout design, responsive design, and interactive web elements.
